Application is build with Bootstrap and is fully responsive.

#2 Gibbet Game

A simple game called hangman. Player can choose one out of two categories and three difficulty levels. The game generates a number of lives to use based on difficulty level. Application written in EcmaScript 6 using Webpack. It is a browser-based game that does not use a database. All the passwords needed to play are stored as a jSON object, so there is a possibility to extend the application to the back-end. The site is fully responsive.
